# LinkedIDontKnowWhatToReplyAI

Browser extension for **Chrome** and **Edge** that gives suggestions for replies to **LinkedIn** posts via **OpenAI**'s **chatgpt-4o**. An **OpenAI Api Key** is required to use it.
